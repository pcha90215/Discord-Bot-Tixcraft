Command:
!help   Shows this message
!start  開始執行監控任務
!status 查看目前的設定狀態與網址
!stop   暫停監控任務
!test   測試機器人是否能在當前頻道說話
!time   設定檢查頻率 (秒)
!url    設定要監控的拓元網址

操作步驟:
1.將Bot加入Discord伺服器(discord developers -> Installation -> Install Link)
2.更新Token後，丟入VScode執行(Discord developers -> Bot -> Reset Token)
3.在Discord使用command操作
